  <action>
  The user needs to create a new webhook in Microsoft Teams. This cannot be automated — it requires the Teams web/desktop UI.

  **Steps:**
  1. Open Microsoft Teams
  2. Navigate to the channel where DS01 alerts should be posted
  3. Click the "+" button or go to channel settings → "Manage channel"
  4. Search for "Workflows" app (NOT "Incoming Webhook" connector — that's deprecated)
  5. Select the template: **"Post to a channel when a webhook request is received"**
  6. Follow the wizard to create the workflow
  7. Copy the resulting webhook URL — it will look like:
     `https://prod-XX.eastus.logic.azure.com:443/workflows/GUID/triggers/manual/paths/invoke?api-version=...`
  8. Provide the URL to Claude

  **Important:** The old Incoming Webhook connector URLs (format `webhookb2/...`) no longer work as of October 2024. The new format uses Power Automate Workflows via `logic.azure.com`.
  </action>
